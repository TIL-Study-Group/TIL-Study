# Hash 함수 문제풀이  

완주하지 못한 선수 (Level 1)

1. Python 
- collection 사용
```python
//collection 사용 간단하게
import collections
def solution(participant, completion):
    return list((collections.Counter(participant) - collections.Counter(completion)).keys())[0]
```
- Hash함수와 dictionary 사용

```python
def solution(participant, completion):
    temp = 0
    dict = {}
    
    for p in participant:
        dict[hash(p)] = p
        temp += hash(p)
    for c in completion:
        temp -= hash(c)
    
    return dict[temp]
```
